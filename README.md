# students-Attendance-Tracking

The Student Attendance Tracker is a Python-based application developed to simplify and automate the process of managing student attendance. Traditional attendance management methods rely on manual record-keeping, which can be time-consuming, prone to errors, and difficult to maintain. This application provides a digital solution for recording, storing, and tracking student attendance efficiently.
The system is developed using Python, with Tkinter for the graphical user interface (GUI) and CSV file handling for data storage. It allows users to enter student details such as Roll Number and Name, mark attendance status as Present or Absent, and save the records automatically with the current date. The stored attendance data can be viewed, searched, and analyzed through the application's interface.
The application includes features such as attendance recording, attendance viewing, student search functionality, and attendance summary generation. These features help faculty members monitor attendance records accurately and reduce the workload associated with manual attendance management.
The primary objective of this project is to provide an easy-to-use, reliable, and efficient attendance management system that improves data accuracy and accessibility. The project also demonstrates the practical implementation of Python programming concepts, including GUI development, file handling, data management, and event-driven programming.
Key Features
Student attendance recording with date and time.
Present/Absent attendance status selection.
Automatic storage of attendance records in CSV format.
Search attendance records using Roll Number.
View complete attendance history.
Generate attendance summaries and reports.
User-friendly graphical interface using Tkinter.
Efficient and error-free attendance management.
Technologies Used
Programming Language: Python
GUI Framework: Tkinter
Database/Storage: CSV File
Modules Used: csv, tkinter, datetime, os
